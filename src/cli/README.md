# CLI Module

Interactive CLI operations for managing handoff project files.

## Files

| File | Purpose |
|------|---------|
| `create.ts` | `create()` — interactive project initialization wizard |
| `eject.ts` | `ejectConfig()`, `ejectPages()`, `ejectTheme()` — copy template files to user's project |
| `make.ts` | `makeTemplate()`, `makePage()`, `makeComponent()` — scaffold new project files |
| `scaffold.ts` | `runScaffold()` — interactive wizard for scaffolding component stubs from Figma data |
